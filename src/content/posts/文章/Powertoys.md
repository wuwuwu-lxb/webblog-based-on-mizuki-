# 一个提升windows效率的工具
[官方文档在这]([PowerToys Microsoft - Windows 高级用户的实用程序 | Microsoft Learn](https://learn.microsoft.com/zh-cn/windows/powertoys/))
## 下载和安装
**安装要求**
* Windows 10/11 操作系统
* .NET 5.0 或更高版本
* 管理员权限安装
**源**
- [GitHub](Releases · microsoft/PowerToys](https://github.com/microsoft/PowerToys/releases)
- [Microsoft Store](https://apps.microsoft.com/home?hl=zh-CN&gl=CN)
## 功能介绍
powertoys并不是一个toy，相反，它是各个工具的集成。有很多方便的工具 
1. Awake
	保持唤醒，始终不休眠 ,下载东西和后台挂东西的时候比较好用。
2. Color Picker
	取色器
3. FancyZones
	这是窗口布局快照，可以快速布局工作环境，做到一些原本做不到的重叠布局甚至
4. File Locksmith
	删文件或者改动文件的时候，会提醒你有进程在使用了，但你又不知道是什么，这时候，这个工具允许你查看什么进程用到这个文件了
5. Hosts
	管理host？（我不懂哈，应该是本地服务器的ip）
6. Image Resizer
	批量调整图片大小
7. Keyboard Manager
	修改键位映射
8. Mouse Utilities
	改变一点点鼠标的外观和互动方式吧
9. PowerToys Run
	快速启动并且可以输入命令，用多了应该很好用
10. Screen Ruler
	测量像素，感觉用不到
11. 窗口裁剪器
	顾名思义，裁剪窗口，只会取下这么一块，比如你看网页，周遭有广告，或者视频没法全屏，直接把想读的裁剪下来！
12. Always on  top
	使用快捷键实现窗口固定上层或者移除固定
13. 高级粘贴
	实现markdown粘贴，HTML粘贴……
14. 命令面板
	感觉比***开始***更好用，并且有扩展，待开发
15. 环境变量
	管理环境
16. 注册表预览
	可视化注册表
17. 快捷键指南
	想学快捷键？又不知道学啥，看就完了
18. 文本提取器
	好用的OCR
19. 工作区
	比前面的那个布局更强大，好用，推荐
**好多细节没讲，具体用了再说。看这里不如看官方文档和powertoys自己的tutorial**
## 总结
powertoys提供了许多便捷的工具，以及对于一些复杂的可视化，可谓图形界面友好型，把重复性的工作用快捷键和工具替代由此提升效率。
## 使用建议
1. **循序渐进**：不要一次性开启所有功能，先从最需要的2-3个功能开始
2. **自定义配置**：每个工具都提供丰富的设置选项，根据个人习惯调整
3. **快捷键记忆**：掌握主要功能的快捷键，效率提升更明显
4. **定期更新**：PowerToys持续更新，建议保持最新版本以获得最佳体验
